# Visit Rwanda & ProfileCard - React Homework Assignment

This repository contains a React homework assignment demonstrating basic React concepts such as components, props, state, and CSS styling. It consists of **two separate projects**:

## Part 1: Visit Rwanda
A simple React application that displays:
- A heading
- A paragraph
- An image
- A link to an external website

All elements are styled using CSS.

## Part 2: ProfileCard
A reusable React component called `ProfileCard` that:
- Accepts `name` (string), `age` (number), and `isStudent` (boolean) as props
- Displays the userâ€™s name, age, and student status
- Includes a button to toggle the student status between "Student" and "Not a Student"
- Styled with a CSS class for a clean layout

## How to Run

Each project is independent. Make sure you have [Node.js](https://nodejs.org/) installed.

### Steps:
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/visit-rwanda-profilecard.git
   cd visit-rwanda-profilecard
  2. Run Visit rwanda
     cd visit-rwanda
     npm install
     npm run dev

  3.  Run Profile card
      cd ../profilecard
      npm install
      npm run dev
